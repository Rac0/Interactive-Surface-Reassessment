What is Escape From Tarkov?

Escape from Tarkov is a multiplayer first-person shooter video game in development by Battlestate Games for Windows. A closed alpha test of the game was first made available to select users on August 4, 2016, followed by a closed beta which has been running since July 2017.

What is this website for?

This website is for new users to the game to help them learn the maps inside of the game. This website will consist of all the current maps avalible in Escape From Tarkov. Throughout each of the maps, locations of the map will be pin pointed to identify where extracts in-game are located with a photo to show where it is and how it looks in-game. Along side extracts will be landmarks on the map and areas where buried cache containers are located.
